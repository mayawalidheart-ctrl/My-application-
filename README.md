# Clinic Appointment Manager

A simple and intuitive web-based application for managing clinic appointments.

## Features

- Patient registration and information management
- Doctor selection interface
- Appointment scheduling with date and time picker
- Appointment confirmation and management
- Clean, professional medical-themed user interface

## How to Use

1. Open the `dr_walid_clinic_compl_1.html` file in any web browser
2. Register as a new patient or log in if you already have an account
3. Select your preferred doctor from the available options
4. Choose an available date and time for your appointment
5. Confirm your appointment details
6. Your appointment will be saved and displayed in the system
